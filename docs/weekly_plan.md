# Weekly Plan

## Week 7 - Start and Planning
- finalize the project idea
- choose the dataset
- define the target clearly
- choose the project track
- decide the main success metric
- set up the GitHub repository
- assign team roles
- write the project proposal

## Week 8 - Data Understanding and Cleaning
- load the dataset
- understand each column
- create a data dictionary
- clean missing values
- remove duplicates if needed
- fix data type issues
- document assumptions and decisions

## Week 9 - EDA and Baseline
- perform exploratory data analysis
- identify important patterns
- create key charts
- prepare features for modeling
- build the baseline model
- record baseline results

## Week 10 - Improvement and Comparison
- improve preprocessing if needed
- try at least one stronger model
- compare results fairly
- document what changed and why

## Week 11 - Final Approach and Storytelling
- choose the final model
- finalize evaluation
- prepare clear result tables
- select the best charts
- improve README
- draft presentation slides

## Week 12 - Final Polish and Submission
- clean notebooks
- make notebooks run top to bottom
- finalize README
- finalize slides
- review the full repository
- rehearse presentation
- submit the final GitHub repo link
