# Team Roles

## 1. Project Coordinator
**Owner:** To be assigned  
**Responsibilities:**
- keep the project timeline on track
- organize meetings
- follow up on weekly progress
- communicate with the mentor when needed

## 2. Data Lead
**Owner:** To be assigned  
**Responsibilities:**
- load the dataset
- inspect data quality
- handle missing values and duplicates
- prepare the data dictionary

## 3. EDA and Visualization Lead
**Owner:** To be assigned  
**Responsibilities:**
- create charts and visualizations
- explore patterns in the data
- write clear insights from EDA

## 4. Modeling Lead
**Owner:** To be assigned  
**Responsibilities:**
- build the baseline model
- build improved models
- evaluate results
- compare model performance

## 5. Documentation and Slides Lead
**Owner:** To be assigned  
**Responsibilities:**
- maintain the README
- organize project documentation
- prepare the slide deck
- help structure the final presentation

## Shared Responsibility
All team members are responsible for:
- attending meetings
- contributing weekly
- reviewing major decisions
- helping with the final presentation
